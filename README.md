# Web_Project
Pagina Web para administrar reservaciones de un Hotel

## Pasos de instalacion
1. Clonar el repositorio
2. Posicionarse en la carpeta del repositorio.
3. Instalar dependencias con el comando.
```
npm install
```
4. Ejecutar el proyecto
```
npm run dev
```
5. Visitar localhost:3000 en el navegador
